# Remnawave-Scripts (Русская версия)

![версия](https://img.shields.io/badge/версия-1.4.1-blue)
![сделано-на-bash](https://img.shields.io/badge/сделано%20на-bash-green)
![лицензия](https://img.shields.io/badge/лицензия-MIT-green)

Remnawave-Scripts — это кроссплатформенный Bash-скрипт с интерактивным меню, поддержкой двух языков (RU/EN), генерацией случайных ID, поиском стран с эмодзи-флагами и управлением портами.

Также включает систему самообновления с GitHub и позволяет удалить скрипт (rw-scripts) прямо из меню.

---

## 📌 Возможности

- **Генерация случайного shorts_id** — создание уникальных идентификаторов
- **Поиск стран** — поиск на русском и английском, поддерживаются частичные совпадения
- **Показать эмодзи-флаг** и английское название страны
- **💾 Мониторинг памяти** ⭐ v1.1.0
  - Отображение свободной и занятой оперативной памяти
  - Кроссплатформенная поддержка (Linux, macOS)
- **📈 Интерактивный монитор процессов (htop)** ⭐ v1.1.0
  - Автоматическая установка htop при необходимости
  - Мониторинг процессов, CPU и памяти в реальном времени
- **🔒 Управление портами** ⭐ v1.2.0
  - 🔓 Открытие портов через firewall (UFW/FirewallD/iptables)
  - 🔒 Закрытие портов с удалением из базы
  - ✏️ Редактирование описаний портов
  - 📋 Красивый список всех открытых портов
  - 🛡️ Просмотр статуса firewall
  - 💾 Сохранение данных портов между сеансами
- **📦 Обновление системных пакетов** ⭐ v1.2.3
  - Выполнение `apt update && apt upgrade -y` из меню
  - Подтверждение перед запуском
  - Поддержка apt и apt-get
- **🖥️ Настройка сервера** ⭐ v1.3.0
  - 🔑 Смена SSH порта
  - ⚡ Установка, удаление и управление Hysteria2
- **🛡️ Установка zapret** ⭐ v1.4.0
  - Установка одной командой прямо из меню
- **🔧 Сторонние скрипты** ⭐ НОВОЕ в v1.4.1
  - 🌐 Быстрая установка Remnawave от EGames
  - 🤖 Установка Reshala (Remnawave Bedolaga)
  - Автоматическая обработка переменных окружения (bashrc)
- **Информация о системе** — отображение данных системы через neofetch
- **Переключение языка**: Русский / English
- **Интерактивное CLI-меню**
- **Система обновления через GitHub**
- **Возможность удаления** из системы

## 🚀 Установка

Установка одной командой:

```bash
bash <(curl -s https://raw.githubusercontent.com/detective-noir-dev/Remnawave-Scripts/main/install.sh)
```

Запуск глобально:

```bash
rw-scripts
```

---

## ⚙️ Удаление

Если вам больше не нужен **Remnawave Scripts**, вы можете удалить его:

```bash
rw-scripts
# Выберите пункт 4 → 3 (Удалить rw-scripts)
```

Или напрямую:

```bash
~/.local/share/remnawave/uninstall.sh
```

**Подменю управления портами:**

```
╔════════════════════════════════════════════╗
║      🔒 Управление портами / Ports        ║
╚════════════════════════════════════════════╝

1) 🔓 Открыть порт
2) 🔒 Закрыть порт

#### 🔧 Сторонние скрипты (группа 6) ⭐ НОВОЕ в v1.4.1

**Подменю сторонних скриптов:**

```
╔════════════════════════════════════════════╗
║         🔧 Сторонние скрипты              ║
╚════════════════════════════════════════════╝

1) 🌐 Remnawave (EGames)
2) 🤖 Reshala
0) ⬅️  Назад
```

**Что включено:**
- **🌐 Remnawave (EGames):** Автоматическая установка reverse-proxy решения от EGamesAPI. Скрипт сам скачивает установщик, запускает его и корректно обновляет переменные окружения.
- **🤖 Reshala:** Установка популярного скрипта Reshala от DonMatteoVPN. Включает безопасную загрузку во временный файл и автозапуск после установки.

> **⚠️ Важно:** После установки сторонних скриптов может потребоваться перезапуск терминала или выполнение команды `source ~/.bashrc` для активации новых команд. Скрипт автоматически предложит это сделать.

## 📦 Системные требования

- **ОС**: Linux, macOS, BSD, Windows WSL
- **Зависимости** (устанавливаются автоматически при необходимости):
  - `bash` — основной интерпретатор
  - `curl` — для загрузки обновлений
  - `xxd` — для генерации ID (автоустановка)
  - `neofetch` — для информации о системе (опционально)
  - `htop` — для монитора процессов (опционально)
  - `jq` — для работы с JSON в управлении портами (автоустановка)
  - `ufw` / `firewalld` / `iptables` — firewall для управления портами
  - `apt` / `apt-get` — для обновления системных пакетов (Debian/Ubuntu)

## 🔐 Безопасность

**Управление портами:**
- ⚠️ Требуются права sudo для работы с firewall
- 🛡️ Все изменения применяются напрямую к системному firewall
- 💾 Данные портов хранятся локально в вашей домашней директории
- 🔒 Рекомендуется открывать только необходимые порты

**Обновление системных пакетов:**
- ⚠️ Требуются права sudo для установки пакетов
- 📦 Выполняется полное обновление системы
- ⏱️ Процесс может занять значительное время
- 🔄 Рекомендуется выполнять регулярно для безопасности системы

**Сторонние скрипты:**
- ⚠️ Скрипты загружаются из внешних репозиториев (EGamesAPI, DonMatteoVPN)
- 🛡️ Рекомендуется проверять исходный код сторонних скриптов перед запуском
- 🔒 Используйте на свой страх и риск
